Tested up to: 3.9.1
Stable tag: 0.1

Adds support for http(s)://(www.)mixcloud.com/ to the internal oEmbed parser

== Description ==

Whitelists MixCloud so that you can use oEmbed endpoints instead of relying on out-of-date plugins.

== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip the contents of the zip file
2. Upload the `mixcloud-oembed` directory to the `/wp-content/plugins/` directory on your website
3. Activate the plugin through the 'Plugins' menu in WordPress

Any links to Mixcloud content that can embed now will, to force embeding you can wrap the link in [embed]linkhere[/embed]
